Project Idea

Overview

I’ve always valued privacy ever since I learned how easy it was to be removed with technology, as such my project will be a collection of security programs and will serve as a secure hub of sorts containing a VPN, a password manager, an anti-virus, a secure web browser and a secure cloud based encrypted file storage. These will all be accessible through one program i.e. the hub but won’t necessarily all be able to run in said program window and may open as a separate window. It will also allow account management so that you can manage your cloud storage, payment and personal information.

Motivation

In a world that is rapidly evolving with technology becoming more and more integrated into people’s lives, it is becoming quickly apparent that the lines drawn decades ago about an individual’s right to privacy are eroding. In response to data becoming much more valuable to companies that can profit from it, most of the population’s willingness to give it to them in the form of social media privacy and governments using technology to allow them to spy on their citizens it’s become apparent that people have to take their privacy into their own hands.

Description

To begin with I’m going to talk about account management. Right off the bat it might seem counter-intuitive to want a secure service to protect your privacy only to give it away to the person saying they can give you your privacy back but the account is only for 2 things. First it’s there to identify you not personally only by email or username so that if you need to install it onto a new computer since the base program will be a one time purchase and will include everything but the cloud file storage however will have the structures to allow those competent to set up a local secure server and the expanded VPN servers that are more likely to get around a government or companies attempts to stop proxy’s. The reason for this is because servers for both proxies and file storage can get expensive very quickly and without constant cash flow from those that wish to use them the service would quickly sink. The password manager would be local or cloud based with no extra charge but a simple storage limit given that for the most part most people would only need a max of ~15 passwords and since they are simple text they won’t take up much space and given that it's a choice not all will be comfortable having their passwords stored elsewhere. It will also have the ability to generate unique passwords itself that are made of random text made locally so that there is no worry about interception in any way. The VPN or Virtual Private Network functions as a proxy allowing you to essentially trick websites into thinking you’re somewhere else which allows you to either get access to content you couldn’t otherwise or stop your internet provider or someone intercepting your traffic from being able to see exactly what you do. The base version will include access to a few VPN servers in most countries that have modern internet infrastructure allowing access to content in other countries however being limited to a few servers means that it isn’t too hard to be blocked by governments or companies. The subscription model will give access to constantly updated servers in an effort to avoid being blocked and allowing all content to be accessed and watched. The anti-virus will function like other market leaders providing from files or programs hidden in websites, emails or downloads that are likely to harm your computer. The base program will have the ability to scan your computer and quarantine and delete flagged items that it detects while the paid version will let you protect up to 5 computers with real-time flagging of spam or websites that are likely to contain malware as well as parental controls that will allow parents to easily set active hours for all apps or services such as internet as well as restrict some completely. It will also allow you to encrypt folders and work with the password manager to make sure that only you and those you want to share access to are able to get into these folders.

Tools and Technologies

To start with this program is going to need quite a lot of servers. Proxy servers in most countries as well as storage servers to allow people to have their cloud encrypted storage as well as password storage. They have to be reliable and fast servers otherwise they will grow dissatisfied with the software and abandon it since most people leave a site if it takes more than 3 seconds to load. The software required will be fairly extensive including a password manager which can easily found open source, a VPN can be found open source but given what’s required making one is probably more effective, anti-virus can be found open-source and likely expanded to make what’s required, integrating Brave would be easiest since its a nice open source browser that runs well and building a secure file storage system for both the service itself and the password storage.

Skills Required

As mentioned above, fast servers will be required for both the VPN tunnels as well as the file storage because no one will want to use a slow VPN given the amount of options available and secure cloud storage is pointless if just putting the files on a portable hard-drive or USB is faster. Since a lot of the software used is open-source most of the work is done it just requires them to be integrated with each other which is why it’s likely best to build the VPN and file storage in house to make it easier to integrate password storage and make sure the VPN works as intended and given how prevalent most of these services are skills and hardware should be easy and relatively cheap to come by. 

Outcome

With any luck the program will launch and the launch day bugs will get quickly fixed allowing people to have a centralised hub of security services and software that will allow them to not only better protect their personal lives but also better access the world around them without fear of tracking or regional locking. Though if this program were to somehow become big enough or widespread it may only make the people who oppose it such as advertisement companies and governments which seek to control what their citizens see double down on their efforts to suppress or nullify the software.
